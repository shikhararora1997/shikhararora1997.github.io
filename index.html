Portfolio

fotoranker
Flask application

Fotoranker, a sophisticated and fun Flask application, has been designed to rank images and automate photography hackathons and has been used as a first round judge for photography contests held at Thapar. It uses machine learning to sort images and intelligently arrange images in order of their aesthetic appeal based on their aesthetic appeal using real-time human ranking surveys!

DTC - Delhi Transport Commission

Back in my sophomore year, I worked on a cool project called DTC that helped me improve my Flask skills. The project involved using the Folium library in Python to visualize the bus routes between two stops in New Delhi. It was really interesting to work on, and I used the most up-to-date data available at that time, which was from 2017. Although it's been a while, I'm planning to revisit the project soon in order to optimize the code and increase its geographical access, so it can be even more useful and visually appealing!

GitHub DTC

COVID-19 India Dashboard


During my first term in the Master's program, I created an immersive Tableau dashboard that visualized the trends of the COVID-19 outbreak in India. This captivating dashboard was built using publicly available data sourced from Kaggle, providing users with a powerful tool for studying the impact of the pandemic on the country. Its user-friendly interface allows anyone to quickly grasp the severity of the situation and track the progression of the disease over time.
